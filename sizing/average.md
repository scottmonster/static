#### General Mobile Layout and Print Size

Use the following equivalent dimensions for an average modern mobile-device layout.

| Context | Width | Height |
| --- | --- | --- |
| Web/CSS viewport | 406 CSS px | 893 CSS px |
| PDF page | 304.5 pt | 669.75 pt |
| PDF page, rounded | 305 pt | 670 pt |
| Custom paper size | 4.229 in | 9.302 in |

##### CSS Layout

```
406 x 893 px
```

##### PDF Page Size

Use the rounded page size when a whole-number value is required:

```
305 x 670 pt
```

##### Custom Paper Settings

```
Width:  4.229 in
Height: 9.302 in
```

More precise values:

```
Width:  4.229167 in
Height: 9.302083 in
```

Recommended print settings:

```
Paper size: 4.229 x 9.302 in
Margins: 0
Scale: 100%
Headers and footers: Off
Orientation: Portrait
```
