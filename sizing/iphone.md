# iPhone Layout and Print Size

Use the following equivalent dimensions for an iPhone-sized layout.

| Context | Width | Height |
| --- | ---: | ---: |
| Web/CSS viewport | 413 CSS px | 895 CSS px |
| PDF page | 309.75 pt | 671.25 pt |
| PDF page, rounded | 310 pt | 671 pt |
| Custom paper size | 4.302 in | 9.323 in |

## CSS Layout

```text
413 x 895 px
```

## PDF Page Size

Use the rounded page size when a whole-number value is required:

```text
310 x 671 pt
```

## Custom Paper Settings

```text
Width:  4.30 in
Height: 9.32 in
```

More precise values:

```text
Width:  4.302 in
Height: 9.323 in
```

Recommended print settings:

```text
Paper size: 4.30 x 9.32 in
Margins: 0
Scale: 100%
Headers and footers: Off
Orientation: Portrait
```

## Conversion Reference

```text
96 CSS px = 72 pt
1 CSS px = 0.75 pt

413 / 96 = 4.302 in
895 / 96 = 9.323 in
```
